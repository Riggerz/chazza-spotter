# Chazza Spotter v1

A simple Flask phone-friendly prototype with Jacko Brain verdicts.

## Run
```bash
pip install flask
python app.py
```
Open `http://127.0.0.1:5000` on your computer, or use your computer IP on your phone while on the same Wi-Fi.

## What it does now
- Brand tier lookup
- Buy / maybe / check hard / leave verdict
- Max buy price
- Expected resale list range
- Saved spots in `spots.json`
